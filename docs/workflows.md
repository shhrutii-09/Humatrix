# Leave Workflow

Employee applies leave  
↓  
HR reviews request  
↓  
HR approves/rejects leave

If HR applies leave:  
↓  
Organization Admin approves/rejects

# Attendance Workflow

Employee checks in  
↓  
Attendance record created  
↓  
Employee checks out  
↓  
Total hours calculated  
↓  
HR/Admin monitors attendance

# Attendance Correction Workflow

Employee submits correction request  
↓  
HR reviews correction  
↓  
HR approves/rejects

# Overtime Workflow

Employee submits overtime request  
↓  
HR reviews request  
↓  
HR approves/rejects

If HR submits overtime:  
↓  
Organization Admin reviews request

# Task Workflow

HR creates task  
↓  
Task assigned to employee  
↓  
Employee updates task status  
↓  
HR monitors progress

# Notification Workflow

System generates notifications for:

- leave approvals
- overtime approvals
- task assignments
- attendance corrections
